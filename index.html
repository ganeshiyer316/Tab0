Tab Age Tracker - Chrome Extension

Welcome to the Tab Age Tracker extension! This tool helps you manage and monitor your browser tabs with detailed analytics and insights.

Download the Extension

Get the latest version of the Tab Age Tracker Chrome extension:

Download Extension (ZIP)

Installation Instructions:

  1. Download and unzip the file
  2. Open Chrome and go to chrome://extensions
  3. Enable "Developer mode" (toggle switch in top-right)
  4. Click "Load unpacked" and select the unzipped folder
  5. The extension will appear in your browser toolbar
