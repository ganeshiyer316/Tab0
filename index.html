Tab Age Tracker Chrome Extension

This extension helps you track and manage your browser tabs by monitoring their age and providing analytics to help reduce tab clutter.

Download Extension v1.9 (.zip)

Installation Instructions

  1. Download the extension zip file using the button above
  2. Extract the zip file to a folder on your computer
  3. Open Chrome and navigate to chrome://extensions
  4. Enable "Developer mode" by toggling the switch in the top right corner
  5. Click "Load unpacked" and select the folder where you extracted the extension files
  6. The Tab Age Tracker extension should now be installed and visible in your browser toolbar

Key Features

Tab Age Tracking

Tracks how long each tab has been open and categorizes them by age (Today, This Week, This Month, Older)

Visual Analytics

Interactive charts and visualizations to help you understand your tab usage patterns

Smart Grouping

Suggests tab groups based on website domains, content similarity, and age

Progress Tracking

Shows your progress toward reaching "Tab Zero" based on your peak tab count

URL Date Extraction

Intelligently extracts dates from URLs to improve age detection accuracy for tabs without direct creation timestamps
