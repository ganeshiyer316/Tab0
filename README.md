# TabSearch

A fast, privacy-focused Chrome extension for searching and managing your browser tabs.

## Features

- **Instant Search** - Find any tab by title or URL
- **Tab Age Tracking** - See how old each tab is (Today, 1-7 days, 8-30 days, 30+ days)
- **Domain Grouping** - View tabs organized by website
- **Duplicate Detection** - Find and close duplicate tabs with one click
- **Optional Notifications** - Get alerts for tabs older than your threshold
- **Dark Mode** - Automatic dark/light theme based on system preference

## Privacy

**100% Local** - TabSearch stores all data locally on your device. No data is ever sent to external servers.

- No analytics or tracking
- No network requests
- No account required
- Your browsing data stays on your device

[Read our full Privacy Policy](https://ganeshiyer316.github.io/Tab0/privacy-policy.html)

## Permissions

| Permission | Purpose |
|------------|---------|
| `tabs` | Read tab titles, URLs, and favicons for search functionality |
| `storage` | Store tab timestamps and user preferences locally |
| `alarms` | Periodic badge updates and old tab checks |
| `notifications` | Optional alerts for old tabs |

### Manual Installation
1. Download or clone this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode" (toggle in top right)
4. Click "Load unpacked" and select the extension folder

## Usage

1. Click the TabSearch icon in your toolbar
2. Type in the search box to find tabs
3. Click age categories to filter by tab age
4. Click domain names to see tabs from that site
5. Use the settings gear to customize preferences
