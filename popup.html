About Tab Age

Tabs are categorized by age to help you identify which ones you've been keeping open for too long.

  • Opened Today - Opened in the last 24 hours
  • Open 1-7 Days - Opened between 1-7 days ago
  • Open 8-30 Days - Opened between 8-30 days ago
  • Open >30 Days - Opened more than 30 days ago
  • Unknown Age - Tabs that existed before extension installation

For accuracy, tabs that existed before the extension was installed are marked as "Unknown Age". Only new tabs opened after installation have verified age data.
