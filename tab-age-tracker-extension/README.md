# Tab Age Tracker - Chrome Extension

## Version 1.2

This Chrome extension helps you track and manage your browser tabs by monitoring their age and providing insights to reduce tab clutter.

## Features

- **Tab Age Tracking**: Monitors and displays the age of all your open tabs
- **Tab Analytics**: Visual dashboards showing tab trends over time
- **Smart Notifications**: Alerts for tabs that have been open for too long
- **Tab Groups Suggestions**: Intelligent grouping recommendations based on URL patterns and tab age
- **Daily Progress Tracking**: Charts showing your progress towards tab zero
- **Web Dashboard**: Access comprehensive analytics via the web dashboard

## Privacy First

- All sensitive tab data remains local to your browser
- Only aggregated metrics are optionally shared with the web dashboard (with your consent)
- No personally identifiable information is collected

## Requirements

- Chrome browser (or Chromium-based browser) version 88 or higher

## Installation

1. Unzip this file to a folder on your computer
2. Open Chrome and navigate to `chrome://extensions/`
3. Enable "Developer mode" using the toggle in the top-right corner
4. Click "Load unpacked" and select the folder where you extracted the extension
5. The extension is now installed and will begin tracking your tabs

## Usage

- Click the extension icon to see a quick overview of your tabs
- Visit the options page for detailed analytics and settings
- Enable web dashboard sync in settings to access your analytics online
