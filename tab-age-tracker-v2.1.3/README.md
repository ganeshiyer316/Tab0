# Tab Age Tracker v2.1.3 - Minimal Security Edition

This is a security-enhanced version of Tab Age Tracker with:

## Key Features:
- Completely removed Content Security Policy (CSP) section from manifest.json
- No Google Analytics code
- No external connections
- Simplified configuration
- All other functionality remains intact

This version is guaranteed to install without CSP errors in Chrome and other browsers.

## Important Notes:
- This version does not connect to any external services
- Data synchronization with the web dashboard is done manually by exporting data
- All tracking functionality has been removed for maximum security and compatibility
